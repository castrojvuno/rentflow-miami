# RentFlow Miami — RoomControl Pro

Sistema de gestión de propiedades de alquiler compartido.

## Estructura
```
rentflow/
├── index.html
├── package.json
├── vite.config.js
├── vercel.json
└── src/
    ├── main.jsx
    └── App.jsx
```

## Para correr localmente
```bash
npm install
npm run dev
```

## Para publicar en Vercel
1. Sube esta carpeta a GitHub
2. Importa el repositorio en vercel.com
3. Vercel detecta Vite automáticamente
4. Click en Deploy
